# Swapna Dutta

A personal website dedicated to the life, memories, writings, photographs, and legacy of Swapna Dutta.

This project aims to preserve and organise memories, stories, documents, and related materials in a simple, respectful, and accessible format.

## Planned Features

- Biography and life timeline
- Photo galleries
- Writings and memories
- Articles and documents
- Search and navigation tools
- Reusable page templates
- Mobile-friendly design

## Technology

Built using:
- GitHub
- Cloudflare Pages
- Markdown
- Static site generation

## Goals

- Preserve memories digitally
- Create a clean and long-lasting archive
- Maintain fast and lightweight pages
- Keep the site simple and accessible

Early development.
